ЗАГРУЗКА КЛЮЧА ЭЦП НА НОВЫЕ iPhone/iPad

Если у вас мобильное устройство с новой IOS, которая не блокирует сохранение файлов внутрь iPhone/iPad - для загрузки ключа на iPhone/iPad вам необходимо открыть на компьютере программу  
iCloud Drive и загрузить ключ внутрь программы, например в папку DOWNLOADS, после чего вы можете выбирать загруженный ключ прямо с вашего iPhone/iPad. Как дополнительный вариант - используйте инструкцию как для Apple-устройств старого типа.

ЗАГРУЗКА КЛЮЧА НА СТАРЫЕ iPhone/iPad. Часть 1. Передача ключа на ваше устройство

Если у вас устройство Apple (iPhone/iPad):
1. Зайдите на эту же страницу с вашего компьютера, где вы храните свой ключ подписи

2. Выберите ваш ключ по кнопке "Обзор...":
Это может быть:
- ключ для входа в сервис (имя ключа для авторизации начинается с AUTH_RSA256...)
- ключ для подписи ИП/физлица (имя ключа для подписи начинается с RSA256...)
- ключ для подписи юридического лица (имя ключа для подписи начинается с GOST...)
3. После выбора файла система покажет вам ссылку, скопируйте ссылку и отошлите ссылку сами себе на свою почту
4. ВНИМАНИЕ! Не открывайте ссылку из почты нажатием, скопируйте ссылку в буфер обмена и из буфера обмена скопируйте в браузер вашего телефона, после чего просто откройте ссылку в браузере телефона и перейдите к шагу 2 на экране
